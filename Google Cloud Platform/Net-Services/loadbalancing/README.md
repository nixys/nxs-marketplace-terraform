# loadbalancing

## Introduction

This is a set of terraform modules for the Google Cloud Platform provider for building a Net-Services and creating any different loadbalancer's resources.

## Features

- Supported grpc target, https target, http target, ssl target, tcp target
- Supported health-cheks
- Supported globale forwarding rules
- Supported forwarding rules
- Supported backend service
- Supported grpc region target, https region target, http region target
